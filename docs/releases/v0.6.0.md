# v0.6.0 — LangGraph Runtime and Durable Tool Replay

Release date: 2026-07-11 UTC

## Scope

Module 4 / Task 06 production Agent graph runtime: compile locked JSON topologies
into LangGraph `StateGraph`, preserve dependency-free reference execution, and
prove checkpoint / interrupt / MCP replay boundaries without company algorithms.

## Delivered

- Locked JSON main graph + two subgraphs compiled by a production adapter
- TypedDict/AgentState checkpoint conversion with size/JSON/raw-byte guards
- Synchronous durability, isolated thread IDs, interrupt/resume
- Model, RAG and MCP node adapters with opaque Artifact handles
- SQLite MCP replay ledger (`PENDING` / `COMPLETED` / `UNCERTAIN`)
- PostgreSQL checkpointer integration and CI service wiring
- Cross-runtime same-thread advisory lock / lease protocol
- Trusted Artifact Registry validation for remote MCP I/O
- Real-LangGraph success/failure matrix and 20-way concurrent isolation
- Process-kill window after ledger complete / before graph checkpoint

## Explicit non-claims

- Company MCS parsing, TensorRT, planning and safety algorithms are not
  reimplemented; only injectable ports and deterministic local backends exist
- Container replace, host crash, PostgreSQL SIGKILL/WAL recovery and network
  partition remain field evidence
- Tool internal side effects vs ledger commit atomicity remains a production
  backend responsibility

## Local verification baseline

- `python3 run_tests.py` on the demo host
- Gated real-LangGraph / PostgreSQL tests run in CI when services are present
