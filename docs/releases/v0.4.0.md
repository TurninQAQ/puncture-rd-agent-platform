# v0.4.0 — Local Enterprise Hybrid RAG Demo

Release date: 2026-07-11 UTC

## Scope

Module 2 replaces the explicit production RAG stub with a locally runnable,
dependency-free enterprise retrieval pipeline and optional provider adapters.
The default demo is intentionally deterministic: it needs no OpenSearch process,
model download, GPU, network connection or company document.

This release proves the RAG control flow and provider protocols offline. Live
OpenSearch, Qwen embedding/reranker inference and production-corpus quality or
latency evidence are `NOT_RUN` and are not claimed.

## Implemented

- heading-aware Markdown ingestion with parent/child chunks, coherent tables/code,
  bounded mixed-CJK splitting and embedding batches;
- stable document/parent/chunk identity, checksums, immutable version handling,
  explicit updates and atomic in-memory rebuild generations;
- deterministic local BM25 and dense recall, Reciprocal Rank Fusion, reranking,
  parent-context expansion, context budget and normalized citations;
- fail-closed ACL, module, exact-version and lifecycle filters in each recall branch,
  parent lookup and final output;
- strict request/config types, total retrieval deadline, normalized errors,
  hashed-by-default trace queries and best-effort trace export;
- offline evaluation for Recall@5/10, MRR, NDCG@10, correct-version hit rate,
  ACL leak count, no-answer accuracy and P50/P95 latency;
- secure pooled HTTP transport, OpenSearch BM25/filtered-kNN adapter,
  OpenAI-compatible Qwen embedding adapter and vLLM `/v1/rerank` adapter;
- OpenSearch 3.7.0 mapping/bootstrap/smoke/snapshot/alias assets with complete
  embedding/parser/chunker manifest metadata;
- generic EDA, fab-equipment and engineering-change-control local demo.

## Local execution

```bash
python3 examples/local_rag_demo.py
python3 -m unittest discover -s tests/rag -p 'test_*.py' -v
python3 run_tests.py
```

The demo ingests four synthetic industrial documents, returns cited evidence for
an authorized query, and returns zero evidence for an exact restricted identifier
when the caller lacks its scope.

## Explicit limitations

- no live OpenSearch container, alias promotion, snapshot restore or TLS handshake
  was executed for this release;
- no Qwen embedding or reranker weights were downloaded or served;
- deterministic local embeddings/reranking prove orchestration, not semantic quality;
- no internal 80–150 query Golden Set was available, so production Recall/NDCG,
  ACL-leak and P50/P95 claims are absent;
- no real company document or proprietary algorithm is committed;
- wheel clean-install smoke is `NOT_RUN` because the demo host has no `pip`/`wheel`
  command; source imports, compilation and executable demos pass locally;
- the next modules still need MCP production adapters, LangGraph runtime,
  checkpoint/recovery API and integrated Trace/Eval.

The interview-safe statement is: the repository contains a complete local Hybrid
RAG demo and offline-tested production adapters; live service and real-corpus
measurements require separate deployment evidence.
